# TrackWise Frontend

This is the frontend service for the TrackWise Personal Asset Management System, built with **React**, **TypeScript**, and **Vite**.

## Developer Cheatsheet

Below is a complete list of `npm` commands you will use during frontend development. All commands should be run from this `frontend/` directory.

### 1. Installation

Before running anything, make sure all Node dependencies are installed:

```bash
npm install
```

### 2. Running the Application Locally

To start the Vite development server with hot-module replacement (runs on `http://localhost:5173`):

```bash
npm run dev
```

### 3. Linting

To run ESLint and check for code quality or syntax issues:

```bash
npm run lint
```

### 4. Building for Production

To compile the TypeScript code and build an optimized production bundle:

```bash
npm run build
```

_(The generated static files will be located in the `dist/` directory)._

### 5. Previewing the Production Build

If you want to test the production build locally before deploying:

```bash
npm run preview
```
